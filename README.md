## Instalação das dependências
Os exemplos serão usando o npm
```bash
$ npm install
```

## Inicialização do back end
```bash
# development
$ npm run start

# watch mode
$ npm run start:dev
```

## Inicialização do front end
```bash
# development
npm run dev
```

## Tecnologias e libs utilizadas
### front end:
- Vue js 3
- Vuetify
- Pinia
- Vue Router

### back end
- Nest js
- Mysql
- Supabase
